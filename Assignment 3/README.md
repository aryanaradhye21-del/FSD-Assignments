# 🏛️ MODA | Curated Fashion Atelier

MODA is a high-end, minimalist e-commerce prototype designed for a premium user experience. This project demonstrates clean UI/UX principles, dynamic DOM manipulation, and responsive web design without the use of heavy frameworks.

## ✨ Key Features

- 🛒 Dynamic Shopping Bag: Add and remove items with real-time price calculation and a sleek slide-out sidebar.
- 🔍 Smart Filtering: Instantly sort the collection by category (Outerwear, Tops, Bottoms) using JavaScript.
- 📱 Ultra-Responsive: Hand-crafted CSS Grid and Flexbox layouts that adapt perfectly from 4K monitors to mobile devices.
- ✨ Micro-Interactions: Smooth hover effects, sticky navigation transitions, and slide-in animations.
- 💎 Premium Aesthetic: Modern typography using Google Fonts and high-contrast minimalist styling.

## 🛠️ Technical Breakdown

### Core Technologies
- HTML: Semantic structure for SEO and accessibility.
- CSS3: Custom properties (variables), Grid, and Flexbox for layout logic.
- JavaScript : Functional programming used for state management and DOM rendering.
